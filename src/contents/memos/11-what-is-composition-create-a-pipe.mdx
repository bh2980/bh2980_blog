## 문제

함수를 연결하여 합성하는 pipe 함수를 만들어주세요.

## 풀이

<Collapsible>
  ```ts
  function pipe(funcs) {
    return function(arg){
      return funcs.reduce((acc, func) => {
        return func.call(this, acc)
      }, arg)
    }
  }
  ```
</Collapsible>
