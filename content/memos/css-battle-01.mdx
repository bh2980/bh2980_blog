# CSS Battle #1 - Simply Square

## 문제

200x200 오렌지색 정사각형을 화면 중앙에 배치

## 내 풀이 (60 characters)

```html
<div></div>
<style>
  div {
    width: 200px;
    height: 200px;
    background: #b5e0ba;
    margin: 50px 100px;
  }
</style>
```

## 더 짧은 풀이 (45 characters)

```html
<style>
  * {
    margin: 50px 100px;
    background: #b5e0ba;
    width: 200px;
    height: 200px;
  }
</style>
```

## 배운 점

- margin을 사용한 중앙 정렬
- CSS 단축 속성 활용
