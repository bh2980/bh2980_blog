```js !code
function debounce(func, wait) {
  let timer = null;
// !tooltip[/debounced/] debounced
  return function debounced(...args){
// !tooltip[/clearTimeout/] clearTimeout
    clearTimeout(timer);
// !tooltip[/setTimeout/] setTimeout
    timer = setTimeout(() => func.call(this, ...args), wait);
  }
}
```
```

## !!tooltips debounced

debounced는 실제로 호출되는 함수이므로 this를 바인딩해줘야해 function 키워드를 사용해야 한다.
```
