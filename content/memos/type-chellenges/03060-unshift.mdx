## **3060 - Unshift**

> `Array.unshift`의 타입 버전을 구현하세요.

```typescript
type Result = Unshift<[1, 2], 0>; // [0, 1, 2,]
```

## **풀이**

```typescript
type Unshift<T, U> = T extends [...infer Rest] ? [U, ...Rest] : never;
```
