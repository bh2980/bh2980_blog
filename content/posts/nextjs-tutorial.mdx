# Next.js 시작하기

Next.js는 React 기반의 풀스택 프레임워크입니다.

## 주요 특징

- **Server-side Rendering (SSR)**: 서버에서 HTML을 생성하여 SEO와 초기 로딩 성능을 개선
- **Static Site Generation (SSG)**: 빌드 타임에 정적 페이지를 생성하여 빠른 로딩 속도 제공
- **App Router**: 새로운 라우팅 시스템으로 더 직관적이고 강력한 기능 제공
- **API Routes**: 백엔드 API를 쉽게 구축할 수 있는 기능

## 설치 방법

```bash
npx create-next-app@latest my-app
cd my-app
npm run dev
```

## App Router 기본 구조

App Router는 `app` 디렉토리 내에서 파일 시스템 기반으로 라우팅을 제공합니다.

```
app/
├── layout.tsx       # 루트 레이아웃
├── page.tsx         # 홈 페이지
├── about/
│   └── page.tsx     # /about 페이지
└── blog/
    ├── page.tsx     # /blog 페이지
    └── [slug]/
        └── page.tsx # /blog/[slug] 동적 페이지
```

## 서버 컴포넌트 vs 클라이언트 컴포넌트

- **서버 컴포넌트**: 기본값, 서버에서 렌더링되어 번들 크기 감소
- **클라이언트 컴포넌트**: `"use client"` 지시어 사용, 상호작용이 필요할 때
