원래 블로그 게시글이 SSG로 동작하지 않고 있다는 것은 알고 있었습니다. 하지만 코드 블럭 에디터 쪽 로직 개발이 바빠서 본문 쪽에서 사용 중인 `searchParams`만 없애면 동작하겠거니 라는 낙관적인 생각에 방치해왔습니다.&#x20;

그런데 에디터 코드 블럭의 작동 방식을 변경하면서 Mermaid 렌더링을 빌드 타임에서 해줘야 할 필요성이 생겼습니다. 원래는 클라이언트 쪽에서 렌더링을 했는데, 변경 이후에 렌더링 로직을 클라이언트까지 끌고가니 라이브러리 번들링 과정에서 뭔가가 꼬였는지 계속 `Internal Server Error`가 계속 나오더라구요. 결국 이를 해결하기 전에 SSG로 전환해주는 작업에 들어가기로 했습니다.
