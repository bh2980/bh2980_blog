# CSS Grid 완벽 가이드

CSS Grid는 2차원 레이아웃을 위한 강력한 도구입니다.

## 기본 개념

- **grid-container**: `display: grid`가 적용된 부모 요소
- **grid-item**: grid-container의 직접적인 자식 요소
- **grid-line**: 그리드를 구분하는 선
- **grid-track**: 두 그리드 라인 사이의 공간 (행 또는 열)
- **grid-cell**: 가장 작은 그리드 단위
- **grid-area**: 하나 이상의 셀로 구성된 직사각형 영역

## 컨테이너 속성

### 1. 그리드 정의

```css
.container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr; /* 3개 열 */
  grid-template-rows: 100px auto 50px; /* 3개 행 */
  gap: 20px; /* 간격 */
}
```

### 2. 반복 함수 활용

```css
.container {
  /* 동일한 크기 반복 */
  grid-template-columns: repeat(3, 1fr);

  /* 최소/최대 크기 설정 */
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
}
```

### 3. 명시적 이름 지정

```css
.container {
  grid-template-areas:
    "header header header"
    "sidebar main main"
    "footer footer footer";
}

.header {
  grid-area: header;
}
.sidebar {
  grid-area: sidebar;
}
.main {
  grid-area: main;
}
.footer {
  grid-area: footer;
}
```

## 아이템 속성

### 1. 위치 지정

```css
.item {
  grid-column: 1 / 3; /* 1번 라인부터 3번 라인까지 */
  grid-row: 2 / 4;

  /* 또는 span 사용 */
  grid-column: span 2; /* 2개 열 차지 */
}
```

### 2. 정렬

```css
.container {
  justify-items: center; /* 수평 정렬 */
  align-items: center; /* 수직 정렬 */
  place-items: center; /* 둘 다 */
}

.item {
  justify-self: end; /* 개별 아이템 수평 정렬 */
  align-self: start; /* 개별 아이템 수직 정렬 */
}
```

## 실제 활용 예제

### 반응형 카드 레이아웃

```css
.card-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  gap: 2rem;
  padding: 2rem;
}
```

### 성배 레이아웃

```css
.holy-grail {
  display: grid;
  grid-template:
    "header header header" auto
    "nav main aside" 1fr
    "footer footer footer" auto / 200px 1fr 200px;
  min-height: 100vh;
}
```

CSS Grid를 활용하면 복잡한 레이아웃도 간단하고 직관적으로 구현할 수 있습니다! 🎯
