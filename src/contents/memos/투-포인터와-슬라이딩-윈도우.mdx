## 투 포인터

투 포인터는 두 개의 포인터를 사용해서 탐색 범위를 줄여가는 기법이다.\
같은 방향애서 시작할 때는 start = 0, end = 1로 시작하고, 양 끝에서 시작할 때는 0, len() - 1에서 시작하면 편하다.

## 슬라이딩 윈도우

슬라이딩 윈도우는 \[start, end]인지 \[start, end)인지 명확하게 정하고 코드를 정한다.\
코드를 짤 때는 항상 슬라이딩 윈도우 내부가 유효한 상태를 유지하도록 코드를 짜는 것이 좋다.
