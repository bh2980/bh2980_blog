# Tailwind CSS 유용한 팁들

Tailwind CSS 사용 시 도움이 되는 팁들을 소개합니다.

## 반응형 디자인

- `sm:`, `md:`, `lg:`, `xl:`, `2xl:` 접두사 활용
- 모바일 퍼스트 접근법 (기본 스타일은 모바일 기준)

```html
<div class="text-sm md:text-base lg:text-lg">반응형 텍스트 크기</div>
```

## 유용한 유틸리티 클래스들

### 1. 공간 설정

```html
<!-- 균등한 간격 -->
<div class="space-y-4">
  <div>아이템 1</div>
  <div>아이템 2</div>
</div>

<!-- 그리드 간격 -->
<div class="grid grid-cols-3 gap-4">
  <!-- 그리드 아이템들 -->
</div>
```

### 2. 텍스트 제한

```html
<!-- 한 줄 제한 -->
<p class="truncate">매우 긴 텍스트...</p>

<!-- 여러 줄 제한 -->
<p class="line-clamp-3">여러 줄의 긴 텍스트...</p>
```

### 3. 호버 효과

```html
<button class="bg-blue-500 hover:bg-blue-600 transition-colors">
  부드러운 호버 효과
</button>
```

## 커스텀 설정

### tailwind.config.js 활용

```javascript
module.exports = {
  theme: {
    extend: {
      colors: {
        brand: "#your-color",
      },
      spacing: {
        72: "18rem",
        84: "21rem",
      },
    },
  },
};
```

## 성능 최적화

### 1. PurgeCSS 설정

```javascript
module.exports = {
  content: ["./src/**/*.{js,ts,jsx,tsx,mdx}"],
  // ...
};
```

### 2. 자주 사용하는 패턴은 @apply로 추출

```css
@layer components {
  .btn-primary {
    @apply px-4 py-2 bg-blue-500 text-white rounded hover:bg-blue-600;
  }
}
```

Tailwind CSS의 유틸리티 퍼스트 접근법을 활용하여 빠르고 일관성 있는 스타일링을 경험해보세요! ✨
