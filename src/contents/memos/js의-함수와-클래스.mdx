## 함수

JS의 함수는 일급 객체로 함수 선언문 또는 함수 표현식으로 정의할 수 있다.

### 함수 선언문

```ts
add(1, 2); // 3

function add(x, y){
	return x + y;
}
```

함수 선언문은 function 키워드로 함수의 이름을 명시하여 독립적으로 선언하는 방식이다. 자바스크립트 엔진이 코드를 실행하기 전 [호이스팅](https://bh2980.dev/memos/js%EC%9D%98-%EC%BD%94%EB%93%9C-%EC%8B%A4%ED%96%89-%EB%A9%94%EC%BB%A4%EB%8B%88%EC%A6%98#%ED%98%B8%EC%9D%B4%EC%8A%A4%ED%8C%85)을 통해 함수 선언을 먼저 메모리에 로드하기 때문에 함수 선언보다 먼저 함수를 호출 할 수 있습니다.

### 함수 표현식

```ts
add(1, 2); // TypeError
sub(2, 1); // ReferenceError

var add = function (x, y){
	return x + y;
}

const sub = function (x, y){
	return x - y;
}
```

함수 표현식은 함수 리터럴을 변수에 할당하는 방식이다. 때문에 변수 선언만 호이스팅되고 실제 함수 할당은 코드가 해당 줄에 도달했을 때(런타임) 이루어지므로, 선언 이전에 호출 시 에러가 발생한다.

### 일급 객체와 함수

일급 객체란 어떤 객체를 일반적인 값과 동일하게 취급할 수 있음을 나타내는 개념이다. 일급 객체가 되기 위해서는 아래와 같은 조건을 만족해야한다.

* 변수나 객체의 프로퍼티 값에 할당할 수 있어야한다
* 함수의 매개변수로 전달할 수 있어야한다
* 함수의 반환 값으로 사용될 수 있어야한다
* 무명의 리터럴로 생성할 수 있어야한다.

JS에서 함수는 일급 객체로 이러한 특성 덕에 고차 함수나 클로저와 같은 패턴을 활용할 수 있다.
