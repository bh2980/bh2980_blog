# CSS 가상 요소 활용법

## 기본 문법

```css
.element::before {
  content: "";
  display: block;
  /* 스타일 */
}
```

## 활용 예시

1. 아이콘 추가
2. 장식 요소 생성
3. 레이아웃 보조
4. 호버 효과

## 주의사항

- content 속성 필수
- display 속성 설정 권장
