```js !code
function debounce(func, wait) {
let timer = null;
// !tooltip[/debounced/] debounced
return function debounced(...args){
// !tooltip[/clearTimeout/] clearTimeout
clearTimeout(timer);
// !tooltip[/setTimeout/] setTimeout
timer = setTimeout(() => func.call(this, ...args), wait);
}
}
```

## !!tooltips debounced

debounced는 “호출 시점의 this”를 그대로 받아서 func에 넘겨야 하므로, 화살표 함수가 아니라 function을 써서 동적 this를 받는다.
